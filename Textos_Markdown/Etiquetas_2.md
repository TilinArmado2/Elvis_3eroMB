

## Resumen de Etiquetas formularios

`<form>`: Contenedor principal que agrupa todos los campos de entrada.

`<label>`: Proporciona una etiqueta de texto vinculada a un campo específico.

`<input type="text">`: Campo para ingresar texto simple (Nombre, Apellido, CI).

`<input type="email">`: Campo especializado para direcciones de correo electrónico.

`<input type="tel">`: Campo para números de teléfono con optimización móvil.

`<input type="password">`: Campo que oculta los caracteres ingresados por seguridad.

`<input type="file">`: Permite al usuario seleccionar y subir archivos (CV).

`<input type="checkbox">`: Casillero de selección para aceptar términos o contacto.

`<select>`: Menú desplegable para elegir entre varias opciones predefinidas.

`<option>`: Define cada una de las alternativas dentro de un menú desplegable.

`<textarea>`: Área de texto de múltiples líneas para descripciones extensas.

`<button type="submit">`: Botón encargado de procesar y enviar el formulario.

## Resumen de etiquetas de validacion y control

`required`: Obliga al usuario a completar el campo antes de enviar.

`placeholder`: Muestra un texto sugerido o de ayuda dentro del campo vacío.

`pattern`: Define una expresión regular para validar el formato (solo números).

`maxlength`: Limita la cantidad máxima de caracteres permitidos (ej. 8 para CI).

`accept`: Especifica los tipos de archivos permitidos en una carga (ej. .pdf).

`autocomplete`: Indica al navegador que sugiera datos guardados del usuario.

`inputmode`: Determina qué teclado mostrar en dispositivos móviles (ej. numérico).
