# Etiquetas relevantes 
- `<!DOCTYPE html>`: Etiqueta principal, indica que es una pagina de tipo HTML.
- `<lang>`: El atributo lang en HTML define el idioma natural del contenido de una página

# Texto

`<h1>` Título principal (el más importante)

`<p>` Párrafo de texto

`<br>` Salto de línea

`<strong>` Texto en negrita (importante)

`<em>` Texto en cursiva (énfasis)

# Tablas
`<table>`  Crear una tabla

`<tr>` Fila de la tabla

`<td>` Celda normal

`<th>` Celda de encabezado

# Formularios

`<form>` Formulario

`<input>` Campo de entrada (texto, email, etc.)

`<button>` Botón

`<label>` Texto que describe un input
