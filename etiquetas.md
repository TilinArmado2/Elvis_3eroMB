## etiquetas necesarias

# Estructura Básica
- `<!DOCTYPE html>`: Define que es un documento HTML5.
- `<html>`: Raíz del documento.
- `<head>`: Contiene metadatos, título y enlaces a estilos.
- `<body>`: Contiene todo el contenido visible de la página. 

# Contenido y Estructura

- `<h1>` a `<h6>`: Encabezados o títulos, donde `<h1>` es el más importante.
- `<p>`: Define un párrafo.
- `<a>`: Crea hiperenlaces (links).
- `<img>`: Inserta imágenes (etiqueta autocerrada).
- `<div>`: Contenedor genérico para agrupar elementos.
- `<span>`: Contenedor en línea para texto o elementos pequeños. 

# Listas

- `<ul>`: Lista desordenada (con viñetas).
- `<ol>`: Lista ordenada (numerada).
- `<li>`: Elemento dentro de una lista (`<ul>` o `<ol>`). 

# Etiquetas de Formato

- `<strong>`: Texto importante (generalmente negrita).
- `<em>`: Texto enfatizado (generalmente cursiva).
- `<br>`: Salto de línea.
- `<hr>`: Línea horizontal temática. 
# Etiquetas relevantes 
- `<!DOCTYPE html>`: Etiqueta principal, indica que es una pagina de tipo HTML.
- `<lang>`: El atributo lang en HTML define el idioma natural del contenido de una página

# Texto

`<h1>` Título principal (el más importante)

`<p>` Párrafo de texto

`<br>` Salto de línea

`<strong>` Texto en negrita (importante)

`<em>` Texto en cursiva (énfasis)

# Tablas
`<table>`  Crear una tabla

`<tr>` Fila de la tabla

`<td>` Celda normal

`<th>` Celda de encabezado

# Formularios

`<form>` Formulario

`<input>` Campo de entrada (texto, email, etc.)

`<button>` Botón

`<label>` Texto que describe un input
