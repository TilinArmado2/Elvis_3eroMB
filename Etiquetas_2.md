

## Resumen de Etiquetas

| Elemento | Propósito | Atributos Críticos |
| :--- | :--- | :--- |
| `<input type="tel">` | Teléfono | `pattern`, `inputmode` |
| `<input type="file">` | Archivos | `accept=".pdf"` |
| `<input type="password">` | Seguridad | `required` |
| `<textarea>` | Texto largo | `placeholder` |
| `autocomplete` | UX / Rapidez | `given-name`, `family-name`, `tel` |
| `required` | Integridad | N/A |
